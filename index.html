Offline File Drop
Checking network…

Move files with no internet at all.

One laptop turns on its hotspot, the other connects to it. From there, this page links the two directly — no server, no upload, no size limit. One side creates a room, the other joins with a short code you read off the screen.

How this works offline

  1. Connect to the same network. Turn on a hotspot on one laptop, connect the other to it. No internet needed on either.
  2. Laptop A creates a room and gets a short connection code.
  3. Laptop B joins by typing in that code, and gets a reply code back.
  4. Laptop A finishes by entering the reply code. Now you're linked, device to device.
  5. Drag files in. They stream straight across — no size cap, no waiting room on a server.
